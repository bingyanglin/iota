# Getting Started with Wasm

## Minimum Requirements

-   [Node.js](https://nodejs.org/en) (>= `v20`)

## Install the Library

If your project does not contain it already, install the peer dependency `@iota/iota-sdk` as well.

```bash npm2yarn
npm install @iota/iota-sdk
```

You can install the latest beta version of the library by running the following command:

```bash npm2yarn
npm install @iota/notarization@alpha
```

## Build the Library

Alternatively, you can build the bindings to work with currently unreleased features. You can find instructions for this in the [`notarization` repository](https://github.com/iotaledger/notarization/tree/v0.1/bindings/wasm/notarization_wasm#build).
