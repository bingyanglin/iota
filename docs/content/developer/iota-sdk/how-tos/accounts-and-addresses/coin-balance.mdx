# Get Coin Balance

import IotaSdk from '../../../../_snippets/developer/iota-sdk/sdk.mdx';
import Nanos from '../../../../_snippets/developer/iota-sdk/nanos.mdx';

<IotaSdk />

This guide shows you how to retrieve the coin balance for an IOTA address using the IOTA SDK's [GraphQL client](/developer/references/iota-graphql).

## Prerequisites

- The IOTA SDK installed for your target language
- A valid IOTA address on the testnet

## Steps

### 1. Initialize the GraphQL Client
